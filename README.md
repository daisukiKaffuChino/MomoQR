## 简介

MomoQR 是一款简单的二维码扫描和生成工具，为老旧备用机打造。

## 开发背景

某天，正在使用搭载~~低版本~~类原生的 Xperia XZ1 的我发现了一个难题：

如何在不翻墙找软件的情况下，不使用任何国产垃圾应用扫码？...

于是，我倒了一杯蓝山并打开电脑， `MomoQR` 就这么诞生了。

## 注意

不再积极开发，因为项目的完善程度已基本满足开发者个人的要求。

由于发现本仓库源代码被倒卖并被抹去来源信息，后续提交暂不公开。
