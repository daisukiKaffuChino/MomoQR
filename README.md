## Introduction

**MomoQR** is a modern QR code management tool that supports scanning, generating, and centrally managing all your QR codes.

## ⚠️ Breaking Changes in Progress

The project is currently undergoing a **complete rewrite**, migrating to **Jetpack Compose**.

This means:

- The code structure will change significantly
- Some features may be temporarily unavailable
- Versions may not be fully compatible

After the rewrite, the project will provide:

- A more modern UI
- A cleaner architecture
- Better maintainability
- Improved scalability
