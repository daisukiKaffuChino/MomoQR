## 简介

MomoQR是一款简单的二维码扫描和生成工具，为老旧备用机打造。

*缓慢开发中*

## 开发背景

某天，正在使用搭载类原生的 Sony XZ1 的我发现了一个难题：

如何在不翻墙找软件的情况下，不使用任何国产垃圾应用扫码？

于是，我倒了一杯蓝山并打开电脑， `MomoQR` 就此诞生了。
